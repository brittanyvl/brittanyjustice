# Brittany Justice | Portfolio

Personal portfolio website showcasing data engineering projects and professional experience.

**Live Site:** [brittanyvl.github.io/brittanyjustice](https://brittanyvl.github.io/brittanyjustice)

---

## Quick Start

Open `index.html` in your browser, or use a local server:
```bash
python -m http.server 8000
# Open http://localhost:8000
```

### Deploy to GitHub Pages

1. Go to repository **Settings** > **Pages**
2. Select **Deploy from a branch** > `main` > `/ (root)`
3. Click **Save**

---

## Customization

Edit `index.html` to update content, `css/style.css` to change colors.
